# Obsidian skill examples

This is an accompanying repository for [a guide I released about creating skills for Obsidian](https://ronihelppi.com/writing/a-practical-guide-to-writing-your-own-obsidian-skills/). This repository contains examples of agentic skills I use alongside [Obsidian](https://obsidian.md/). As they are heavily influenced by my own processes and the vault structure, they probably should not be blindly copied. Instead, go read the [accompanying blog post](https://ronihelppi.com/writing/a-practical-guide-to-writing-your-own-obsidian-skills/) to learn how to create skills by yourself, or ask the agent to edit my skills to suit your processes and vault.

[A practical guide to writing your own Obsidian skills — Roni Helppi](https://ronihelppi.com/writing/a-practical-guide-to-writing-your-own-obsidian-skills/)

## Skills

| Skill | Description |
|-------|-------------|
| [obsidian-cli](.claude/skills/obsidian-cli) | Interact with Obsidian vaults via the [Obsidian CLI](https://help.obsidian.md/cli) |
| [obsidian-create-effort](.claude/skills/obsidian-create-effort) | Create effort (= project) notes in a structured way |
| [obsidian-prepare-writing](.claude/skills/obsidian-prepare-writing) | Evaluate notes and prepare them for writing and publication |
| [obsidian-summarize-material](.claude/skills/obsidian-summarize-material) | Create a table of contents -style notes from materials like articles, videos, or podcasts |
| [obsidian-weekly-review](.claude/skills/obsidian-weekly-review) | Run a structured weekly review |
